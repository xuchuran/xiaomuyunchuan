## ✨ 为什么选择小木云传？

### 🚀 **极速传输 - 比传统方式快3000倍**
- **多线程并发上传**：最高支持12线程同时传输
- **智能分块技术**：自动优化块大小（10MB-100MB可选）
- **断点续传**：网络中断？没问题，从断点继续！
- **实时速度监控**：随时掌握传输状态

### 🎨 **炫酷界面 - 视觉盛宴**
- **彩色火箭动画**：渐变色火箭图标，动态脉冲效果
- **现代化设计**：圆角卡片、渐变背景、流畅动画
- **响应式布局**：完美适配桌面和移动设备
- **实时进度显示**：精美的进度条和统计信息

### 📁 **批量上传 - 效率翻倍**
- **拖拽批量上传**：一次选择多个文件，批量处理
- **文件状态管理**：实时显示每个文件的上传状态
- **智能队列**：自动排队，按序上传，支持取消和重试
- **批量统计**：总体进度、平均速度、完成率一目了然

### 🔐 **安全可靠 - 企业级保障**
- **端到端加密**：文件传输全程加密保护
- **多账户支持**：支持演示账户和自定义账户
- **本地存储**：文件安全存储在本地服务器
- **访问令牌**：支持高级用户使用API令牌
